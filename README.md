# Chargerzilla Admin Panel

Professional SaaS-style admin panel for managing EV charging stations.

## Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **React Router** - Navigation
- **Recharts** - Data visualization
- **Lucide React** - Icons

## Features

- ✅ Clean, professional light theme
- ✅ Dashboard with metrics and charts
- ✅ Stations table with search and filters
- ✅ Excel file import with progress tracking
- ✅ Geographic distribution views
- ✅ Settings and configuration

## Installation

```bash
cd chargerzilla-admin
npm install
```

## Development

```bash
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
```

## Design Principles

- Function over form
- Readability first
- No dark theme, no glassmorphism
- Clean data tables
- Professional SaaS aesthetic
